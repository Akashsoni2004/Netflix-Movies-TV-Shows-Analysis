# Netflix-Movies-TV-Shows-Analysis
This project explores a dataset of 10,000+ Netflix movies and TV shows to uncover insights about content trends, genre distribution, country-wise production, and growth patterns over time. The goal is to perform Exploratory Data Analysis (EDA) and derive meaningful, data-driven insights using Python.
🎯 Objectives
Analyze distribution of Movies vs TV Shows
Identify top genres and content trends
Explore country-wise content production
Study yearly growth of Netflix content
Generate actionable insights for content strategy
🛠️ Tech Stack
Python
Pandas & NumPy → Data cleaning & manipulation
Matplotlib & Seaborn → Data visualization
📂 Dataset
Contains 10,000+ records of Netflix titles
Features include:
Title, Type (Movie/TV Show)
Genre, Country, Release Year
Duration, Rating, Date Added
🔍 Key Insights
📈 ~70% of content added after 2015, showing rapid platform growth
🎬 Movies dominate (~65–70%) compared to TV Shows
🌍 USA and India are the top content-producing countries
🎭 Drama & Comedy are the most common genres (~40–50%)
⏰ Peak content addition observed in recent years, indicating aggressive expansion
📊 Visualizations

The project includes 15+ visualizations, such as:

Count plots (content type distribution)
Heatmaps (correlation analysis)
Bar charts (top genres, countries)
Time-series plots (yearly growth trends)
🚀 Key Features
Data cleaning with handling of missing values (~10%)
Feature engineering for better analysis
Insightful visual storytelling using graphs
Beginner-friendly and well-structured code
